# aiw wt

Worktree management for aiw.

## Usage

aiw wt [command] [args...]

## Description

Worktree management for aiw. Supports adding, removing, listing, and pruning worktrees.

## Examples

- `aiw wt add task1`
- `aiw wt rm task1`
- `aiw wt list`
- `aiw wt prune`
- `aiw vc release app.c /Feapp.exe`
- `aiw vc debug app.c /Feapp_dbg.exe`
