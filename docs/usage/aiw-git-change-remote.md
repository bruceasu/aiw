# aiw git-change-remote

Short: Change the remote for the current branch.

Description:
Changes the remote for the current branch.

Usage:
aiw git-change-remote <url> [remote]

Arguments:
- <url> — The URL of the repository to change the remote to.
- [remote] — The name of the remote (default: origin).

Examples:
- aiw git-change-remote https://github.com/user/repo.git origin
